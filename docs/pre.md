# 기능 요구사항

## JSON 라이브러리

- JSON 데이터 파싱 / JSON 파일 저장을 담당하는 라이브러리를 사용해 CRUD를 구현한다.
  - [JSON 파싱](FEATURES/json-parsing.md)
  - [JSON 파일 저장](FEATURES/json-file-storage.md)

## CRUD 기능 구현

- **Create**: 새로운 데이터를 입력받아 JSON 파일에 저장한다. → [상세](FEATURES/create.md)
- **Read**: 전체 목록 보기 및 특정 ID/키 값으로 검색하는 기능을 제공한다. → [상세](FEATURES/read.md)
- **Update**: 기존 데이터를 선택하여 특정 필드를 수정한다. → [상세](FEATURES/update.md)
- **Delete**: 특정 데이터를 안전하게 삭제한다. → [상세](FEATURES/delete.md)
