# Delete

## 개요

특정 회원 데이터를 안전하게 삭제하는 기능.

## 동작

- 사용자로부터 삭제할 대상의 ID(또는 키 값)를 입력받아 [read](read.md) 검색 기능으로 대상을 찾는다.
- 대상이 존재하지 않으면 삭제 불가를 알리고 종료한다.
- 대상이 존재하면 삭제 전 확인 절차(예: 대상 정보 출력 후 Y/N 확인)를 거쳐 실수로 인한
  삭제를 방지한다.
- 확인되면 메모리 상의 회원 목록에서 해당 데이터를 제거한다.
- [json-file-storage](json-file-storage.md)를 호출해 변경된 목록을 JSON 파일에 저장한다.

## 관련 기능

- [read](read.md): 삭제 대상을 찾을 때 사용.
- [json-file-storage](json-file-storage.md): 삭제 결과를 파일에 반영.
