# Read

## 개요

저장된 회원 데이터를 조회하는 기능. 전체 목록 보기와 특정 ID/키 값 검색을 지원한다.

## 동작

### 전체 목록 보기

- [json-parsing](json-parsing.md)으로 로드된 회원 목록 전체를 콘솔에 출력한다.
- 데이터가 없을 경우 빈 목록임을 알린다.

### 특정 ID/키 값 검색

- 사용자로부터 검색할 ID(또는 키 값)를 입력받는다.
- 회원 목록에서 해당 ID와 일치하는 데이터를 찾아 출력한다.
- 일치하는 데이터가 없을 경우 찾을 수 없음을 알린다.

## 관련 기능

- [json-parsing](json-parsing.md): 조회 대상 데이터를 파일에서 불러올 때 사용.
- [update](update.md), [delete](delete.md): 검색된 항목을 대상으로 후속 작업을 수행할 때 재사용.
